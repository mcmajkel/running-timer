# 🏃 Bieg Timer

Interwałowy timer biegowy z GPS. PWA — działa na iPhone przez Safari.

## Funkcje
- Plan 12-tygodniowy (cel: 5 km)
- Sygnały dźwiękowe przy zmianie tempa
- GPS — dystans i tempo w czasie rzeczywistym
- Ekran nie wygasa podczas treningu (Wake Lock)
- Neonowy motyw kolorystyczny per faza

## Lokalny development

```bash
npm install
npm run dev
```

## Deploy

Push na `main` → automatyczny deploy na GitHub Pages przez GitHub Actions.

## Użycie na iPhone

Otwórz w Safari → Udostępnij → Dodaj do ekranu głównego.
